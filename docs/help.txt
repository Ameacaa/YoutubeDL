DESCRIPTION:
    Download best resolution audio or video from youtube playlist or video
COMMAND USE:
    ytbdl.py [/dltype] [link]
ARGUMENTS:
    /dltype         Type you want to download (video/audio) Default: Audio
    link            The link from youtube (video/playlist)
USAGE:
    python ytbdl.py default     -- This command use the default link in ytbdl.py file
    python ytbdl.py link
    python ytbdl.py video link
